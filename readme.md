# OhMyFood

Site web de réservation de restaurants, entièrement codé from scratch en HTML/CSS.

## Comment le démarrer

Ce projet est un site statique HTML/CSS. Pour le lancer, il est recommandé d'utiliser un service comme l'extension Live Server dans VSCode :

- Installez l'extension Live Server dans VSCode si ce n'est pas déjà fait.
- Ouvrez le dossier du projet dans VSCode.
- Clic droit sur index.html et sélectionnez "Open with Live Server". Cela lancera un serveur local (généralement sur http://127.0.0.1:5500)
